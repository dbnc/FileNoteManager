## 一、项目概述

File Note Manager（FNM）是一款 Windows 桌面应用程序，允许用户通过右键上下文菜单为任意文件或文件夹附加文本备注、标签，并提供搜索、历史、导出等功能。

## 四、核心模块说明

### 4.1 数据层 (`Data/`)

| 类 | 职责 |
|----|------|
| `DatabaseManager` | 管理 SQLite 连接池，提供 Query/Execute 方法 |
| `DatabaseInitializer` | 创建表结构、索引、触发器（幂等，使用 IF NOT EXISTS） |

**数据库文件位置**: `%AppData%\FileNoteManager\fnm.db`

**FTS5 全文搜索**: `NoteFTS` 虚拟表，自动通过触发器与 `NoteEntity` 同步。

### 4.2 实体层 (`Repositories/`)

> ⚠️ **注意**: 根目录下的实体文件存在命名混乱问题（历史遗留），文件名与内容不对应，但不影响编译。请参考下表：

| 文件 | 实际包含的类 |
|------|-------------|
| `Repositories/HistoryRepository.cs` | `NoteEntity` |
| `Repositories/BackupRepository.cs` | `HistoryEntity` |
| `Repositories/NoteRepository.cs` | `BackupRecordEntity` |
| `Repositories/TagRepository.cs` | `TagEntity` |

Repository 实现位于 `Repositories/Implementation/`，接口位于 `Repositories/Interfaces/`。

### 4.3 服务层 (`Services/`)

| 服务 | 职责 |
|------|------|
| `NoteService` | CRUD 备注，管理标签关联，自动保存历史 |
| `TagService` | 标签查询，解析标签字符串 |
| `SearchService` | FTS5 全文搜索，路径/日期范围搜索 |
| `HistoryService` | 版本历史读取，版本恢复 |
| `BackupService` | 数据库备份/还原，自动备份策略 |
| `ImportExportService` | JSON/CSV 格式导入导出 |
| `StatsService` | 统计摘要，每日趋势，热门标签 |
| `FileWatcherService` | 监听文件系统变化，自动更新路径引用 |
| `BatchService` | 批量操作备注 |

### 4.4 UI 层 (`FileNoteManager.UI/`)

采用 **MVVM 模式**，使用 `CommunityToolkit.Mvvm` 源生成器。

| 组件 | 职责 |
|------|------|
| `App.xaml.cs` | DI 容器初始化，命令行参数解析 |
| `MainWindow` | 主窗口宿主 |
| `MainViewModel` | 主界面逻辑：备注列表、编辑器、搜索、统计 |
| `NoteListItemViewModel` | 列表项展示模型 |
| `QuickNoteWindow` | 由 Shell 扩展调用时的快捷编辑窗口 |

### 4.5 Shell 层 (`FileNoteManager.Shell/`)

采用 **Windows 注册表静态菜单** 方案（非 COM Shell 扩展）。

```
HKEY_CLASSES_ROOT\*\shell\FileNoteManager
  (Default) = "编辑文件备注(&N)"
  Icon      = "<EXE路径>,0"
HKEY_CLASSES_ROOT\*\shell\FileNoteManager\command
  (Default) = "<EXE路径> --path %1"

HKEY_CLASSES_ROOT\Directory\shell\FileNoteManager  (同上，用于文件夹)
HKEY_CLASSES_ROOT\Directory\Background\shell\FileNoteManager
```

**优势**: 无需 COM 注册，兼容 .NET 8，无需管理员权限（HKCU 方式）。

## 七、数据流示例

### 用户右键文件 → 添加备注
```
Windows Explorer
  → ShellRegistrar 注册的命令
  → 启动 FileNoteManager.UI.exe --path "C:\file.txt"
  → App.xaml.cs 解析参数
  → 创建 QuickNoteWindow
  → QuickNoteWindow 调用 INoteService.GetNote / CreateNote
  → NoteRepository 写入 SQLite
  → FTS5 触发器同步 NoteFTS
```

### 用户在主界面搜索
```
MainViewModel.SearchText 变化
  → ISearchService.Search(keyword, options)
  → SearchService 查询 NoteFTS (FTS5 MATCH)
  → 返回 SearchResult[]
  → MainViewModel 更新 Notes 集合
  → ListView 更新显示
```
