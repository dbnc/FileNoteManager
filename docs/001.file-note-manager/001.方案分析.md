### 安装步骤

#### 方式 1：官网下载安装（推荐）
1. 访问：https://dotnet.microsoft.com/download/dotnet/8.0
2. 下载 .NET 8.0 SDK（Windows x64）
3. 运行安装程序
4. 重启终端/命令行窗口
5. 验证安装：`dotnet --version`

#### 方式 2：使用 winget 安装
```powershell
winget install Microsoft.DotNet.SDK.8
```

#### 方式 3：使用 Chocolatey 安装
```powershell
choco install dotnet-sdk
```

### 安装后验证
```powershell
# 检查 SDK 版本
dotnet --version

# 检查已安装的 SDK
dotnet --list-sdks

# 检查运行时
dotnet --list-runtimes
```

## 执行步骤

### 阶段 1：环境准备
1. 安装 .NET 8.0 SDK
2. 验证安装成功
3. 配置 NuGet 源（如需要）

### 阶段 2：项目创建（需要用户确认后执行）
1. 创建解决方案文件
   ```powershell
   dotnet new sln -n FileNoteManager
   ```

2. 创建项目结构
   ```
   FileNoteManager/
   ├── src/
   │   ├── FileNoteManager.Core/         # 核心库
   │   ├── FileNoteManager.Shell/        # Shell Extension
   │   └── FileNoteManager.UI/           # WPF UI
   └── tests/
       ├── FileNoteManager.Core.Tests/   # 单元测试
       └── FileNoteManager.PropertyTests/ # 属性测试
   ```

3. 创建项目文件
   ```powershell
   # 核心类库
   dotnet new classlib -n FileNoteManager.Core -o src/FileNoteManager.Core
   
   # Shell Extension（类库）
   dotnet new classlib -n FileNoteManager.Shell -o src/FileNoteManager.Shell
   
   # WPF 应用
   dotnet new wpf -n FileNoteManager.UI -o src/FileNoteManager.UI
   
   # 测试项目
   dotnet new xunit -n FileNoteManager.Core.Tests -o tests/FileNoteManager.Core.Tests
   dotnet new xunit -n FileNoteManager.PropertyTests -o tests/FileNoteManager.PropertyTests
   ```

4. 添加项目到解决方案
   ```powershell
   dotnet sln add src/FileNoteManager.Core/FileNoteManager.Core.csproj
   dotnet sln add src/FileNoteManager.Shell/FileNoteManager.Shell.csproj
   dotnet sln add src/FileNoteManager.UI/FileNoteManager.UI.csproj
   dotnet sln add tests/FileNoteManager.Core.Tests/FileNoteManager.Core.Tests.csproj
   dotnet sln add tests/FileNoteManager.PropertyTests/FileNoteManager.PropertyTests.csproj
   ```

5. 添加项目引用
   ```powershell
   # Shell 引用 Core
   dotnet add src/FileNoteManager.Shell/FileNoteManager.Shell.csproj reference src/FileNoteManager.Core/FileNoteManager.Core.csproj
   
   # UI 引用 Core
   dotnet add src/FileNoteManager.UI/FileNoteManager.UI.csproj reference src/FileNoteManager.Core/FileNoteManager.Core.csproj
   
   # 测试引用 Core
   dotnet add tests/FileNoteManager.Core.Tests/FileNoteManager.Core.Tests.csproj reference src/FileNoteManager.Core/FileNoteManager.Core.csproj
   dotnet add tests/FileNoteManager.PropertyTests/FileNoteManager.PropertyTests.csproj reference src/FileNoteManager.Core/FileNoteManager.Core.csproj
   ```

## 潜在风险

### 风险 1：环境配置问题
- **风险**: .NET SDK 安装失败或版本冲突
- **缓解**: 使用官方安装程序，确保清理旧版本
- **影响**: 无法继续开发

### 风险 2：Shell Extension 权限
- **风险**: Shell Extension 需要管理员权限注册
- **缓解**: 安装程序包含权限提升逻辑
- **影响**: 右键菜单功能无法使用

### 风险 3：数据库文件位置
- **风险**: 数据库文件路径权限问题
- **缓解**: 使用 AppData 文件夹，确保用户有写权限
- **影响**: 无法保存备注数据
