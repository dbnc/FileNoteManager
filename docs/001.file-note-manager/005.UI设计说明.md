# File Note Manager - UI 设计说明

> **文档版本**: 1.0  
> **创建日期**: 2026-05-11  

---

## 一、启动流程

```
FileNoteManager.UI.exe
  │
  ├── 无参数启动 ──────→ 主窗口 (MainWindow)
  │
  └── --path "C:\file.txt" ──→ 快捷编辑窗口 (QuickNoteWindow)
```

`App.xaml.cs` 解析命令行参数，通过 DI 容器创建对应窗口。

```

│──────────────────────────────────────────│
│                                          │
│  [多行文本编辑器，自动获取焦点]            │

│                                          │
│──────────────────────────────────────────│
│  标签: [工作 ×] [重要 ×]                 │
│  添加标签: [___________] [+添加]          │
│──────────────────────────────────────────│
│  [💾 保存并关闭]  [🗑 删除]  [取消]      │

```

### 窗口参数
- **尺寸**: 420 × 340，不可调整大小
- **位置**: 屏幕居中或鼠标附近
- **启动时**: 文本框自动获取焦点
- **Ctrl+Enter**: 快速保存并关闭
- **Escape**: 取消并关闭

## 四、ViewModel 结构

### MainViewModel

```csharp
// 数据属性
ObservableCollection<NoteListItemViewModel> Notes       // 当前显示的备注列表
NoteListItemViewModel? SelectedNote                     // 当前选中的备注
string SearchText                                       // 搜索关键词（实时过滤）
string EditorPath                                       // 编辑器：当前路径（只读显示）
string EditorContent                                    // 编辑器：备注内容（双向绑定）

ObservableCollection<string> EditorTags                 // 编辑器：已添加的标签列表
bool IsEditorEnabled                                    // 是否有选中备注（控制编辑区启用）
string StatusMessage                                    // 状态栏消息
int TotalNotes                                          // 总备注数
string DatabaseSizeDisplay                              // 数据库大小显示
string LastBackupDisplay                                // 最后备份时间显示

// 命令
SaveNoteCommand                                         // 保存当前编辑器内容
DeleteNoteCommand                                       // 删除当前备注
AddTagCommand                                           // 添加标签到编辑器
RemoveTagCommand                                        // 从编辑器移除标签
RefreshCommand                                          // 刷新备注列表
ExportJsonCommand                                       // 导出 JSON
ExportCsvCommand                                        // 导出 CSV
ImportCommand                                           // 导入备注
CreateBackupCommand                                     // 手动备份
RegisterShellCommand                                    // 注册右键菜单
UnregisterShellCommand                                  // 注销右键菜单
```

### NoteListItemViewModel


## 五、数据绑定关键点

### 搜索实时过滤
```
SearchText 变化
  → OnSearchTextChanged(value)
  → 过滤 _allNotes 集合
  → 更新 Notes = 过滤后集合
```

### 选中备注加载编辑器
```
SelectedNote 变化
  → OnSelectedNoteChanged(value)
  → 从 INoteService.GetNote(value.Path) 加载完整数据
  → 更新 EditorPath / EditorContent / EditorTags
  → IsEditorEnabled = true
```

### 保存备注
```
SaveNoteCommand 执行
  → 验证 EditorPath 不为空
  → INoteService.UpdateNote(path, content, tags)  (自动 upsert)
  → 刷新 Notes 列表中对应项的预览
  → 显示 StatusMessage "备注已保存"
```

## 六、Shell 右键菜单集成

### 注册流程（RegisterShellCommand）
```csharp
// ShellRegistrar.Register(exePath) 写入注册表：
HKCU\Software\Classes\*\shell\FileNoteManager
HKCU\Software\Classes\*\shell\FileNoteManager\command
HKCU\Software\Classes\Directory\shell\FileNoteManager  
HKCU\Software\Classes\Directory\shell\FileNoteManager\command
```

### 使用 HKCU（不需要管理员权限）
写入 `HKEY_CURRENT_USER\Software\Classes\` 而非 `HKEY_CLASSES_ROOT\`，  
避免需要管理员权限，对当前用户生效。

### 右键菜单效果
在任意文件/文件夹上右键后，出现：
```
📝 编辑文件备注(&N)
```
点击后执行：
```
FileNoteManager.UI.exe --path "C:\选中的文件路径"
```

## 七、键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存备注 |
| `Ctrl+F` | 聚焦搜索框 |
| `Delete` | 删除选中备注（需焦点在列表时） |
| `Ctrl+Z` | 编辑器撤销 |
| `Escape` | 清除搜索框 / 取消编辑 |
| `F5` | 刷新备注列表 |

## 八、错误提示规范

- **保存成功**: 状态栏显示绿色 "备注已保存" 3 秒
- **保存失败**: 弹出 `MessageBox.Show` 错误对话框
- **删除确认**: 弹出确认对话框，避免误删
- **Shell 注册成功/失败**: `MessageBox.Show` 提示
